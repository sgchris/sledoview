Available Commands

set <key> <value> Create/Update

Set or update a key-value pair in the database

> set user_name "John Doe"
âœ“ Successfully set key 'user_name'
get <key> Read

Retrieve detailed information about a specific key

> get user_name

Size: 8 bytes
Value: John Doe
delete <key> Delete

Delete a key from the database

> delete temp_data
âœ“ Successfully deleted key 'temp_data'
list [pattern] Browse

List keys matching a pattern (supports glob and regex)

> list user_*
Found 3 keys:
  1: user_001
  2: user_123
search <pattern> Search

Search for entries where values match a pattern

> search *@gmail.com

Display total number of records in database

Getting Started

1

Clone the Repository

git clone https://github.com/sgchris/sledoview
cd sledoview
2

Build the Application

cargo build --release
3

Run SledoView

# Linux/macOS
./target/release/sledoview /path/to/your/database.db

# Windows
.\target\release\sledoview.exe C:\path\to\your\database.db

Requirements

  • Rust 1.70+ (2021 edition)
  • Cargo build system
  • SLED database files to work with
