A Sled DB client as an interactive terminal featuring CRUD operations, search patterns, and autocompletion.

Getting Started

Quick Download

Download pre-built executables for immediate use, or build from source for the latest features.

GitHub Releases
1

Clone the Repository

git clone https://github.com/sgchris/sledoview
cd sledoview
2

Build the Application

cargo build --release
3

Run SledoView

# Linux/macOS
./target/release/sledoview /path/to/your/database.db

# Windows
.\target\release\sledoview.exe C:\path\to\your\database.db

Requirements

  • Rust 1.70+ (2021 edition)
  • Cargo build system
  • SLED database files to work with
